# Переводчик (Русский ↔ Меллстройный ↔ USB)

Веб-приложение для перевода между русским, вымышленным «меллстройным» языком и USB‑форматом (латиница / сокращения).  
Словари исключений и фраз подгружаются из внешнего JSON‑файла, что позволяет легко обновлять переводы без изменения кода.

## Возможности

- **Русский → Меллстройный**  
  - Учитываются исключения (например, «училка» → «Учительность», «нормально» → «ну нормально не ну еще посидим еще посидим»).  
  - Для обычных слов работает общее правило: окончание заменяется на **-оность**, **-ёность**, **-ость**, **-есть** или просто добавляется **-ность**.

- **Русский → USB**  
  - Если слово есть в словаре – используется заданная аббревиатура / сокращение (например, «училка» → `UCHILKE`).  
  - Иначе выполняется транслитерация кириллицы в латиницу (например, «привет» → `privet`).

- **Меллстройный → Русский** и **USB → Русский** (обратные переводы)  
  - Работают через те же словари и общие правила.

- Автоматический перевод при вводе текста (debounce 250 мс).
- Кнопка ⇄ для мгновенной смены языков и текста.
- Нельзя выбрать одинаковый язык (автоматическая подмена второго языка).
- Полупрозрачный водяной знак версии (`BETA X.X`) — версию нужно менять вручную в коде (строка `BETA 5.8`).

## Установка и запуск

1. **Склонируйте репозиторий** или скачайте файлы.
2. Убедитесь, что в корневой папке лежат два файла:
   - `index.html` – основной код переводчика.
   - `dictionary.json` – файл со словарями и фразами.
3. Запустите локальный веб‑сервер (например, `npx serve .` или `python -m http.server`) и откройте `index.html`.
4. Готово – переводчик работает полностью автономно, все данные берутся из `dictionary.json`.

## Формат файла `dictionary.json`

```json
{
  "rusToMell": {
    "слово": "перевод на меллстройный",
    "училка": "Учительность",
    ...
  },
  "rusToUsb": {
    "слово": "USB-представление",
    "училка": "UCHILKE",
    ...
  },
  "extraReverse": {
    "друн": "друг",
    "жопотность": "жопа",
    ...
  },
  "extraUsbReverse": {
    "uchilke": "училка",
    "shopа": "жопа",
    ...
  },
  "phrases": {
    "rusToMell": [
      ["нормально", "ну нормально не ну еще посидим еще посидим"],
      ...
    ],
    "mellToRus": [
      ["ну нормально не ну еще посидим еще посидим", "нормально"],
      ...
    ],
    "rusToUsb": [
      ["нормально", "NORM"],
      ...
    ],
    "usbToRus": [
      ["norm", "нормально"],
      ...
    ]
  }
}
